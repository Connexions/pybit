PyBit - python Buildd Integration Toolkit.

A new build system using python and RabbitMQ, providing a RESTful HTTP based API.

Learn more »

WebAPI Endpoints (JSON format)

  • Build Boxes (Specify ~/[id] to go to a specific one, and ~/[id]/status or ~/[id]/jobs to see its jobs)
  • Build Jobs (Specify ~/status to see all unfinished jobs, ~/status/[status] to see all of a specified state, or ~/[id] to go to a specific one.
  • Packages (Specify ~/[id] to go to a specific one)
  • Package Instances (Specify ~/[id] to go to a specific one)
  • Package Architectures (Specify ~/[id] to go to a specific one)
  • Job Statuses (Specify ~/[id] to go to a specific one)
  • Software Distributions (Specify ~/[id] to go to a specific one)
  • Package Formats (Specify ~/[id] to go to a specific one)
  • Software Suites (Specify ~/[id] to go to a specific one)
